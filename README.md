# Lolighter

## ChroMapper plugin

A well-known autolighter comes to ChroMapper as a plugin. New and updated features including boost light placement and extended event for specific environment.

### Features

-   Autolighting
-   Downlighting
-   Note Modifier
    -   Bombs
    -   Inverted
    -   Loloppe
    -   Sliders
